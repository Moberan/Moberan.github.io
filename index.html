Welcome to Moberan Open Source.

저희 모베란에서는 여러 프로젝트에서 우리가 자주 사용하지만, 기본 API로 제공하지 않거나, 제공하는 API를 조금더 쉽고 편리하게 사용할 수 있도록하는 컴포넌트 라이브러리를 오픈소스로 공개하여, 쉽고 빠르게 원하는 기능을 구현하는데 도움을 주는것을 목표로 오픈소스를 개발 및 배포합니다.

moc.js

JavaScript Common Component 라이브러리로 좀 더 편하고 쉽게 다양한 기능들을 제공하여,
보다 빠르게 JavaScript로 여러 기능을 구현할 수 있도록 도와주는 유틸리티 기능들을 제공합니다.
